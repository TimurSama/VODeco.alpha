# Тезисный план до релиза v1 и привлечения пользователей/инвесторов

## Продукт
- Онбординг + роль + профиль + публикации.
- Лента, комментарии, модерация.
- Глобус/карта, кликабельные объекты, карточки с данными.
- Миссии/ачивки/рефералы.
- VOD credits (покупка/стейкинг/отчётность).

## Данные и аналитика
- Источники: OSM/USGS/World Bank + подключение новых API.
- Метрики и события: входы, поиск, клики, конверсия в действия.
- Статусы источников, качество, тренды.

## Контент и документы
- Whitepaper + канон синхронизированы.
- Презентация/roadmap актуальны.
- FAQ: VOD/WTR, риски, этапы.

## Growth и партнёрства
- Страницы заявок: партнёры/инвесторы/покупка токенов.
- Воронки: airdrop/рефералы/миссии.
- Лендинги по ролям и сегментам.

## Юридический минимум
- Дисклеймеры, privacy/terms.
- Описание модели без обещаний доходности.

## Техническая готовность
- PostgreSQL + миграции.
- Логи/мониторинг.
- Стабильный деплой (Vercel).
